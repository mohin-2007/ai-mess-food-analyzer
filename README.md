# 📌 Overview

NutriMess AI is a modern AI-powered nutrition analysis platform designed for students, hostel residents, fitness enthusiasts, and health-conscious individuals.

The application allows users to upload food images and receive detailed nutritional insights including:

- Calories
- Protein
- Carbohydrates
- Fat
- Health Score
- Food Detection
- Nutrition Recommendations

The project combines Artificial Intelligence, Web Development, API Integration, Cloud Deployment, and Health Analytics into a single practical application.

# 🚀 Installation

## Clone Repository

git clone https://github.com/mohin-2007/ai-mess-food-analyzer.git

## Open Project

cd ai-mess-food-analyzer

## Install Dependencies

npm install

## Create Environment Variable

Create a file named:

.env.local

Add:

GEMINI_API_KEY=YOUR_GEMINI_API_KEY

## Run Locally

vercel dev

---

# 🌐 Deployment on Vercel

1. Push project to GitHub

2. Import repository into Vercel

3. Add Environment Variable

GEMINI_API_KEY=YOUR_GEMINI_API_KEY

4. Deploy Project

---

# 🔒 Security

Environment variables are excluded using:

.env

.env.local

.vercel

node_modules

API keys are stored securely using Vercel Environment Variables and are never exposed in frontend code.
